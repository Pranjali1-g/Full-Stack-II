# Experiment 2: State in React

## Aim
To understand state management using React Hooks.

## Description
This experiment uses the useState hook to update and manage data dynamically using buttons.

## Technologies Used
- React
- Vite
- JavaScript

## How to Run
1. Install dependencies:
```bash
    npm install
```
2. Start the server:
```bash
    npm run dev
```


## Output
Displays a counter that increases and decreases on button click.

## Learning Outcome
Learned how to manage and update component state using useState.
