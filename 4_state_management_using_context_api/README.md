# Experiment 4: State Management using Context API

## Aim
To understand global state management using React Context API.

## Description
This experiment uses Context API to share data between components without using props.

## Technologies Used
- React
- Vite
- JavaScript

## How to Run
1. Install dependencies:
```bash
    npm install
```

2. Start the server:
```bash
    npm run dev
```

## Output
Displays user information that can be updated globally.

## Learning Outcome
Learned how to create, provide, and consume global state using Context API.
