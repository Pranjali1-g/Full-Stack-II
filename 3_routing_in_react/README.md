# Experiment 3: Routing in React

## Aim
To implement navigation between pages using React Router.

## Description
This experiment demonstrates client-side routing using react-router-dom for navigating between Home, About, and Contact pages.

## Technologies Used
- React
- Vite
- React Router DOM
- JavaScript

## How to Run
1. Install dependencies:
```bash
    npm install
    npm install react-router-dom
```

2. Start the server:
```bash
    npm run dev
```

## Output
Displays multiple pages with navigation links.

## Learning Outcome
Learned how to implement routing and navigation in React.
