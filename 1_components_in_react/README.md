# Experiment 1: Components in React

## Aim
To understand and implement reusable components in React.

## Description
This experiment demonstrates how to create functional components and reuse them inside the main App component.

## Technologies Used
- React
- Vite
- JavaScript
- HTML
- CSS

## How to Run
1. Install dependencies:
```bash
    npm install
```

2. Start the server:
```bash
    npm run dev
```

## Output
Displays a main component and a child component on the browser.

## Learning Outcome
Learned how to create and use reusable React components.
